## Description

We're looking for a Senior Security Engineer to join our team and help improve the security posture of OWASP projects.

## Requirements

- 5+ years of experience in application security
- Strong knowledge of OWASP Top 10
- Experience with security testing tools
- Python and JavaScript expertise
- Open source contribution experience preferred

## Responsibilities

- Conduct security audits of OWASP projects
- Develop security tools and automation
- Mentor junior security engineers
- Contribute to OWASP documentation

## How to Apply

Send your resume and GitHub profile to careers@owasp.org

## Additional Information

This is a fully remote position with flexible working hours. We offer competitive compensation and the opportunity to work on meaningful open source security projects.
